# Diagrama-Sensor-Delugoss
Este nuevo producto, está diseñado para talleres, y cuenta con una interfaz intuitiva en la que el maestro, puede ingresar el modelo, año, y del auto y descripción del sensor, y después hacer click en el botón verde para generar el diagrama donde se dará la descripción de las posibles causas del daño del sensor y las posibles reparaciones,
